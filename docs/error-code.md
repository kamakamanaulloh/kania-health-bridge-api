# Error Code

| Code | Arti |
|---|---|
| BRIDGE_KEY_NOT_CONFIGURED | BRIDGE_KEY belum diubah dari default |
| UNAUTHORIZED_BRIDGE_KEY | Header X-BRIDGE-KEY salah |
| IP_NOT_ALLOWED | IP tidak masuk whitelist |
| VALIDATION_ERROR | Payload kurang/format salah |
| TOKEN_ERROR | Gagal membuat token SATUSEHAT |
| SATUSEHAT_ERROR | Error dari SATUSEHAT |
| BPJS_ERROR | Error dari BPJS |
| ANTROL_ERROR | Error dari Antrol |
